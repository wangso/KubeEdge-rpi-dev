# This repo contains shell scripts to automate installation of KubeEdge Cloud core (any Cloud) and Edge core (Raspberry Pi 4b)

#### 1) "cloud-core-install.sh" contains scripts to install KubeEdge Cloud core
#### 2) "obtain-token.sh" contains scripts to obtain the token required to join edge core to the KubeEdge cluster
#### 3) "edge-core-install.sh" contains scripts to install edge core
#### 4) "cloud-app-install.sh" contains scripts to install example application on the Cloud core

## Follow the above order to do all the installation
## Make sure you follow the notes in the scripts
